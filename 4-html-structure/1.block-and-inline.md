# HTML Elements Reference Cheatsheet

This cheatsheet contains every tag from the provided reference images, categorized by their default display behavior.

---

## 🧱 Block-Level Elements

*Elements that begin on a new line and take up the full width available.*

| Tag            | Description                                               |
| :------------- | :-------------------------------------------------------- |
| `<address>`    | Contact information for the author/owner.                 |
| `<article>`    | Self-contained, independent content.                      |
| `<aside>`      | Content aside from the main page content (sidebar).       |
| `<blockquote>` | A section that is quoted from another source.             |
| `<canvas>`     | Used to draw graphics via scripting (usually JavaScript). |
| `<dd>`         | Description of a term in a description list.              |
| `<div>`        | A generic container for flow content.                     |
| `<dl>`         | A description list.                                       |
| `<dt>`         | A term/name in a description list.                        |
| `<fieldset>`   | Groups related elements in a form.                        |
| `<figcaption>` | A caption for a `<figure>` element.                       |
| `<figure>`     | Self-contained content (like photos/diagrams).            |
| `<footer>`     | A footer for a document or section.                       |
| `<form>`       | An HTML form for user input.                              |
| `<h1> - <h6>`  | HTML headings (Level 1 to 6).                             |
| `<header>`     | A container for introductory content or nav links.        |
| `<hr>`         | A thematic break (horizontal rule).                       |
| `<li>`         | A list item.                                              |
| `<main>`       | The dominant content of the `<body>`.                     |
| `<nav>`        | A section intended for navigation links.                  |
| `<noscript>`   | Alternate content for users that do not support scripts.  |
| `<ol>`         | An ordered (numbered) list.                               |
| `<p>`          | A paragraph.                                              |
| `<pre>`        | Preformatted text.                                        |
| `<section>`    | A generic standalone section of a document.               |
| `<table>`      | Defines a data table.                                     |
| `<tfoot>`      | Groups footer content in a table.                         |
| `<ul>`         | An unordered (bulleted) list.                             |
| `<video>`      | Used to embed video content.                              |

---

## 🔗 Inline Elements

*Elements that do not start on a new line and only take up as much width as necessary.*

| Tag         | Description               | Tag          | Description              |
| :---------- | :------------------------ | :----------- | :----------------------- |
| `<a>`       | Hyperlink                 | `<abbr>`     | Abbreviation             |
| `<acronym>` | Acronym (Legacy)          | `<b>`        | Bold text                |
| `<bdo>`     | Bi-Directional Override   | `<big>`      | Large text (Legacy)      |
| `<br>`      | Line break                | `<button>`   | Clickable button         |
| `<cite>`    | Citation of a work        | `<code>`     | Computer code text       |
| `<dfn>`     | Definition term           | `<em>`       | Emphasized text          |
| `<i>`       | Italic text               | `<img>`      | Image                    |
| `<input>`   | Input control             | `<kbd>`      | Keyboard input           |
| `<label>`   | Label for form element    | `<map>`      | Client-side image map    |
| `<object>`  | Embedded object           | `<output>`   | Calculation result       |
| `<q>`       | Short inline quotation    | `<samp>`     | Sample output            |
| `<script>`  | Client-side script        | `<select>`   | Drop-down list           |
| `<small>`   | Side comments/Small print | `<span>`     | Generic inline container |
| `<strong>`  | Important text            | `<sub>`      | Subscripted text         |
| `<sup>`     | Superscripted text        | `<textarea>` | Multi-line text input    |
| `<time>`    | Date/Time                 | `<tt>`       | Teletype text (Legacy)   |
| `<var>`     | Variable in math/code     |              |                          |

## 🚀 The Creative "Kitchen Sink" Code

This block utilizes **every single tag** listed above in a functional HTML structure.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>The HTML Element Laboratory</title>
</head>
<body>

<header>
    <h1>The <abbr title="HyperText Markup Language">HTML</abbr> Lab</h1>
    <nav>
        <ul>
            <li><a href="#structure">Structural</a></li>
            <li><a href="#interactive">Interactive</a></li>
        </ul>
    </nav>
</header>

<main>
    <article id="structure">
        <section>
            <header><h2>1. Typography & Semantics</h2></header>
            <p>
                In this <b>bold</b> experiment, we use <em>emphasis</em> and <strong>strong</strong> importance. 
                Maybe some <small>small print</small>, a <big>big</big> word, or <i>italics</i>.
                Chemicals like H<sub>2</sub>O or math like <var>x</var><sup>2</sup> are welcome.
                Check the <tt>teletype</tt> for updates.
            </p>
            
            <blockquote>
                <q>To be or not to be,</q> that is the <cite>Shakespearean</cite> question.
            </blockquote>

            <pre>Preformatted text:
    Line 1
    Line 2</pre>

            <address>Written by <a href="mailto:web@example.com">The Coder</a></address>
            <time datetime="2026-03-15">March 15, 2026</time>
        </section>

        <hr>

        <section id="interactive">
            <figure>
                <img src="[https://via.placeholder.com/150](https://via.placeholder.com/150)" alt="Lab Logo" usemap="#labmap">
                <map name="labmap">
                    <area shape="rect" coords="0,0,150,150" alt="Link" href="#">
                </map>
                <figcaption>Fig.1 - An <dfn>Image Map</dfn> container.</figcaption>
            </figure>

            <form action="#">
                <fieldset>
                    <legend>User Interface</legend>
                    <label for="user">Name:</label>
                    <input type="text" id="user" name="user">
                    
                    <select id="options">
                        <option value="1">Option <samp>Alpha</samp></option>
                    </select>

                    <textarea placeholder="Write code here..."></textarea>
                    
                    <button type="button">Click Me!</button>
                    <output name="result">Result: 100%</output>
                </fieldset>
            </form>
        </section>

        <section>
            <h3>Technical Data</h3>
            <canvas id="labCanvas" width="200" height="50" style="border:1px solid #ccc"></canvas>
            
            <table>
                <tfoot><tr><td colspan="2">End of Table</td></tr></tfoot>
                <tbody>
                    <tr><td><kbd>CTRL</kbd> + <kbd>S</kbd></td><td>Save</td></tr>
                </tbody>
            </table>

            <dl>
                <dt>The <bdo dir="rtl">Mirror</bdo></dt>
                <dd>Definition of a reversed string using <acronym title="Bi-Directional Override">BDO</acronym>.</dd>
            </dl>

            <video controls width="250">
                <source src="movie.mp4" type="video/mp4">
            </video>

            <object data="info.pdf" type="application/pdf" width="300" height="200">
                <p>Unable to display embedded object.</p>
            </object>
        </section>
    </article>
</main>

<aside>
    <h4>Extra Info</h4>
    <p>Check the <span>source code</span> for details.</p>
</aside>

<footer>
    <p>&copy; 2026 HTML Masterclass</p>
    <noscript><p>Please enable scripts for full functionality.</p></noscript>
</footer>

<script>
    console.log("Lab initialized.");
</script>

</body>
</html>
```
