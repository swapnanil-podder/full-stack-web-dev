1. Head and Meta Tags

The page head currently includes:

  • <meta charset="UTF-8"> for text encoding
  • <meta name="viewport"> for mobile-friendly sizing
  • <meta name="description"> for search previews
  • <meta name="keywords"> for page metadata
  • <meta name="author"> for author info
  • <title> for the browser tab title
