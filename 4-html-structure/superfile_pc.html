4.1 Block vs Inline Elements

Block elements take up the full width and start on a new line.

I am a DIV (Block)

I am a Paragraph (Block)

Inline elements only take up as much width as necessary.

I am a SPAN (Inline) I am a Link (Inline)

4.2 Containers: Div & Span

<div> is a generic block container. <span> is a generic inline container.

This is a div container. Inside it, we can have colored text using a span.

4.3 HTML Entities

Special characters must be escaped:

  • Less than: < (&lt;)
  • Greater than: > (&gt;)
  • Copyright: © (&copy;)
  • Euro: € (&euro;)
  • Non-breaking space:   (&nbsp;)
  • Ampersand: & (&amp;)

4.5 Semantic Elements

Semantic elements clearly describe their meaning to both the browser and the developer.

<header> Header Section
<nav> Navigation Bar
<main> Main Content
<section> A generic section
<article> Independent content (Blog post, comment)
<aside> Sidebar content
<footer> Footer Section
