Chapter 3.1: HTML Basics (Headings, Paragraphs, Comments)

Headings: <h1> to <h6> define importance. Browsers may support <h7> but it is non-standard.

Paragraphs: <p> for blocks of text. Use <br> for line breaks.

Horizontal Rule: <hr> for thematic breaks.

Chapter 3.3: Hyperlinks (Anchor Tag)

The <a> tag with href attribute.

  • External: <a href="https://google.com" target="_blank">Google</a> (New tab)
  • Local: <a href="new-page.html">Local File</a>
  • Email: <a href="mailto:hello@example.com">Email Us</a>
  • Phone: <a href="tel:+123456789">Call Us</a>

Chapter 3.4: Images

The <img> tag is empty (no closing tag). Key attributes: src (source), alt (alternative text), width, height.

Width/Height can be in pixels (width="200") or percentages (width="50%").

Chapter 3.5: Quotations & Citations

  • <blockquote>: For long quotes. Often indented. Use cite attribute for URL.
  • <q>: For short inline quotes. Browsers add quotation marks.
  • <abbr>: Abbreviation. Use title for full version. (e.g., HTML)
  • <address>: Contact info for author/owner.
  • <cite>: Title of a creative work.
  • <bdo dir="rtl">: Bi-Directional Override (reverses text).

Chapter 4.1: Block vs Inline Elements

Block-level elements: Start on a new line, take full width. Examples: <div>, <h1-h6>, <p>, <li>, <section>, <header>, <footer>.

Inline elements: Do not start on a new line, only take necessary width. Examples: <span>, <a>, <img>, <strong>, <em>, <code>.

Chapter 4.2: Containers (Div & Span)

<div>: Generic block-level container. Used for layout and styling.

<span>: Generic inline-level container. Used for styling parts of text.

This is a <div>. This is a span inside it.

Chapter 4.3: HTML Entities

Reserved characters must be replaced by entities:

  • Less than (<): &lt;
  • Greater than (>): &gt;
  • Ampersand (&): &amp;
  • Copyright (©): &copy;
  • Euro (€): &euro;
  • Non-breaking space: &nbsp;
